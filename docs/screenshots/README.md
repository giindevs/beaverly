# Screenshots

Drop the app screenshots here so they show up in the main README.

The READMEs reference these filenames — use the same names:

| File | Shows | Suggested capture |
| --- | --- | --- |
| `library.png` | The library (grid of book covers) | 1280×800, light theme |
| `reader.png` | The built-in reader (a PDF/EPUB page) | 1280×800 |
| `detail.png` | A book's detail panel (metadata + actions) | 1280×800 |
| `onboarding.png` | The first-run language/intro screen | as shown |

Tips:
- Capture at **1280×800** (or larger, 16:10) for crisp store/README images.
- Consider one **light** and one **dark** theme shot.
- PNG, cropped to just the app window (no desktop background).
- The Microsoft Store wants at least one 1366×768+ screenshot; these work there too.
