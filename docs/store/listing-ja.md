# Beaverly Shelf — ストア掲載情報（日本語）

## 短い説明
電子書籍ライブラリを整理し、PDF・EPUB・CBZをハイライトとメモ付きで読めます。完全オフライン、アカウント登録も不要です。

## 説明
Beaverly Shelfは、読書のためのおだやかでローカル完結型のデスクトップアプリです。あなたの電子書籍ライブラリをすべて自分のパソコンの中に保管し、PDF・EPUB・CBZの本をそのままアプリ内で、ハイライトやメモを添えながら読むことができます。

アカウント登録も、サインアップも、テレメトリ（利用状況の送信）もありません。Beaverly Shelfは完全オフラインで動作します。インターネットに接続するのは、表紙画像の取得など、あなたが自分で選んで実行した操作だけです。

できること：

• ローカルライブラリを作る — 電子書籍（ファイルまたはフォルダ）を取り込み、著者・シリーズ・タグ・コレクションごとに整理できます。
• アプリの中で読む — PDF・EPUB・CBZに対応した、すっきりとした読書モード。目次、テキスト選択、そしてブラウザのような選択ツールバー（コピー、ハイライト、メモ、Web検索）を備えています。
• ハイライトとメモ — 読みながら書き込めます。元のファイルが変更されることは一切ありません。
• 表紙と書誌情報（任意） — 必要なときにだけ、表紙・あらすじ・著者情報をオンラインで取得できます。送信されるのは検索用の文字だけで、ファイルが送られることはなく、保存の前には必ず確認します。
• ライブラリをバックアップ — 同期用のクラウドフォルダなど、好きなフォルダを選んで保存できます。アプリが自動で何かをアップロードすることはありません。
• 好きな言語で読む — 日本語、ポルトガル語、英語、スペイン語、フランス語、ドイツ語、イタリア語、中国語に対応。

プライバシー第一：あなたのライブラリ、読書、メモは、すべてあなたのパソコンの中にとどまります。Beaverlyアカウントは任意で、必須ではありません。

Beaverly Shelfは、より大きなBeaverly読書エコシステムの一部です。詳しくは beaverly.com.br をご覧ください。

## 主な機能
- ローカル電子書籍ライブラリ：著者・シリーズ・タグ・コレクションで整理
- PDF・EPUB・CBZに対応した内蔵ビューア
- ハイライトとメモ — 元のファイルは変更されません
- 目次とテキスト選択を備えた、すっきりした読書モード
- オンラインの表紙・書誌情報を任意で取得（オプトイン、プライバシー配慮）
- 好きなフォルダにライブラリをバックアップ
- 8言語対応のインターフェース
- 完全オフライン — アカウント不要、テレメトリなし
